How can I get my Google API Key ?


  • Create a project/app in https://console.developers.google.com/project
  • Inside your project go to APIs & auth > APIs and turn on the Google+ API
  • Go to APIs & auth > APIs > Credentials > Public API access and click in the CREATE A NEW KEY button.
  • Select the Browser key option and click on the CREATE button
  • After you're done, Copy your API key and paste it in Google Api Key field in Google+ box at WordPress Admin > Arqam

Tumblr

Getting Your Blog Hostname :

Each blog has a unique hostname. The hostname can be standard or custom.

  • Standard hostname: the blog short name + .tumblr.com.

    Example: greentype.tumblr.com

  • Custom hostname: Anything at all, as determined by a DNS CNAME entry.

    Example: www.davidslog.com

Register a new App :

  • Go To Applications page .
  • Click on + Register application button .
  • Fill out the form :

    1. Application name: (requried) enter any name for the Application .
    2. Application website: (optional) enter your site URL
    3. App Store URL: (optional) Keep it blank .
    4. Google Play Store URL: (optional) Keep it blank .
    5. Application description: (optional) Keep it blank .
    6. Administrative contact email: (requried) enter your email .
    7. Default callback URL: (requried) enter your site URL .

  • Enter the OAuth Consumer Key and Secret Key in the Tumblr App info box and click Submit .
  • It will redirect you to Tumblr Allow Access page .. click on Allow button .

How to find your Linkedin company ID?

  1. Open LinkedIn in a new tab.
  2. Search for your company page in LinkedIn.
  3. In your search results, select your company page.
  4. Navigate to the URL at the top of your company page.
  5. Copy the number immediately following 'www.linkedin.com/company/' but before the question mark.
